# Frontend Rules

- Use shadcn/ui components from @/components/ui/
- Style with Tailwind utility classes, no inline styles
- Data fetching via React Query (useQuery, useMutation)
- Form handling with react-hook-form + zod resolver
- Responsive design: mobile-first approach
- Fonts: Playfair Display for headings, Inter for body text
